1. What is DOM and what is the purpose of DOM?

DOM stands for Document Object Model.Dom is an essential part of making websites interactive. It is an interface that allows a programming language to manipulate the content, structure, and style of a website. One of the main purpose of using DOM is that to make a website in dynamic and interactive.

2. How will you select HTML elements using DOM? Name the DOM methods.

We can select HTML elements using dom in many available methods. here are the some methods:

  • Selecting Elements by ID:getElementById()

    Select an element based on its unique 'id' attribute with the 'getElementById()' method

  • Selecting Elements by Class Name:getElementsByClassName()

    The 'getElementsByClassName()' method selects a collection of HTML elements from their 'class' attribute.

  • Selecting Elements by Tag Name:getElementsByTagName()

    Select a collection of HTML elements by 'tag' name using the 'getElementsByTagName()' method.

  • Selecting Elements with CSS Selectors:querySelector()

    The 'querySelector()' method selects an elements from its 'CSS selector'.

  • Selecting Elements with CSS Selectors:querySelectorAll()

    The 'querySelectorAll()' method selects a collection of NodeList from their 'CSS selector'.

3. What is event bubble?

Event bubbling is a method of event propagation in the HTML DOM API when an event is in an element inside another element and both elements have registered a handle to that event. It is a process that starts with the element that triggered the event and then bubbles up to the containing elements in the hierarchy

4. What is a callback function and why will you use it?

A callback function in javascript is a function that is passed as an argument in another function. Which is then invoked inside the outer function to complete some kind of routine or action.
Callbacks are used to make sure that a function is not going to execute before a task is completed but will execute right after the task has completed. It helps us develop asynchronous JavaScript code and keeps us safe from future errors.
